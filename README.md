# Sprint Planner

Sprint Planner - небольшое веб-приложение для планирования спринта, распределения задач по команде и поиска конфликтов по загрузке на таймлайне.

## Что умеет

- планирует спринт по дням от выбранной даты старта;
- группирует участников по командам и ролям;
- разбивает задачи на фазы с отдельными исполнителями и длительностью;
- пропускает выходные при расчете длительности фаз;
- подсвечивает пересечения задач и перегрузку по людям;
- показывает задачи на общем таймлайне;
- хранит данные локально в браузере.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Запуск локально

```bash
npm install
npm run dev
```

После запуска откройте адрес, который покажет Vite в терминале, обычно это `http://localhost:5173`.

## Сборка

```bash
npm run build
npm run preview
```
